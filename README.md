## Mario Game using Python
This is small python mario game for ping hackathon held at Shiv Nadar University, Delhi NCR.

## Dependencies	
* pygame	
* scipy	

## Steps to run the game from command prompt :
$ pip3 install pygame
$ pip install scipy
$ python main.py

## Controls
* Left: Move left  
* Right: Move right  
* Space: Jump  
* Shift: Boost   
* Left/Right Mouseclick: secret
